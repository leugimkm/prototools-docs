prototools is a set of tools that considerably reduce the amount of time spent writing console applications.

Installation¶

prototools is available on PyPi (MIT license) and installation can be performed by running pip.

python -m pip install prototools

To upgrade the package:

python -m pip install prototools --upgrade

Note

It’s recommended to use virtual environment.

Contribution¶

You can contribute with prototools in so many ways (not just coding). Every idea is welcome! You can suggest new features or report a bug when you find it. Every contribution that you made it’ll be mention in this project.
